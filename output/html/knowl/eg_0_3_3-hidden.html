Soit \(A = \{1, 2, 3, 4\}\text{,}\) \(B = \{p \telque p \text{ est premier} \}\text{,}\) \(C = \{5, 7, 9\}\) et \(D = \{\text{nombres entiers positifs pairs}\}\text{.}\) Alors

\begin{align*} A \cap B &= \{2, 3\}\\ B \cap D &= \{2\}\\ A \cup C &= \{1, 2, 3, 4, 5, 7, 9\}\\ A \cap C &= \varnothing \end{align*}

Dans ce dernier cas on voit que les deux ensembles n'ont aucun élément en commun. Dans ce cas, on dit qu'ils sont disjoints.
